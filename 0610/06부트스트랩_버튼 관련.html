08부트스트랩_버튼 관련

버튼은 button태그를 사용하여 클래스명('btn')을 적용한다.

  • 어둡게 : class="btn btn-dark"
  • 흰색 : class="btn btn-light"
  • 파랑 : class="btn btn-primary"
  • 그레이 : class="btn btn-secondary"
  • 초록 : class="btn btn-success"
  • 하늘색 : class="btn btn-info"
  • 노랑 : class="btn btn-warning"
  • 빨강 : class="btn btn-danger"
  • 일반링크 : class="btn btn-link"
info

버튼은 button, input type="button", a 태그요소에도 모두 적용이 가능하다

버튼 테두리만 적용 class="btn-outline-dark"

버튼 크기지정하기 btn-lg(20px), btn-sm(14px)

버튼 비활성화

a요소 비활성화 disabled

a요소에 비활성화할 때는 href속성을 제거하고 disabled속성을 넣어준다

버튼블럭 class="d-grid gap-2" 적용시 반응형버튼으로 가로100%로 설정됨

버튼그룹지정 class="btn-group"

여러 개의 버튼을 묶어 그룹화하거나, 세로로 겹쳐 놓을 수 있다.

버튼그룹 지정시 첫번째 버튼에만 활성화 하기 : active
